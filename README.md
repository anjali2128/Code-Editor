# 📝 Code Editor

A sleek, browser-based code editor built with HTML, CSS, and JavaScript — perfect for writing, previewing, and experimenting with HTML, CSS, and JS code in real time.

![Preview](https://user-images.githubusercontent.com/anjali2128/code-editor-preview.png) <!-- Replace with actual image link if available -->

---

## 🚀 Features

- 🔤 **Live Preview** — Instantly see the output as you type HTML, CSS, or JavaScript
- 🌗 **Dark Mode** — Eye-friendly theme for long coding sessions
- 🧠 **Smart Structure** — Organized layout for clean writing and editing
- 📱 **Responsive** — Works well on desktop, tablet, and mobile devices
- 💾 **LocalStorage Support** — Automatically remembers your last session

## 🔧 How It Works

1. Type code into the respective HTML, CSS, or JS panels.
2. See the result update instantly in the preview pane.
3. Your code is stored locally so you can come back anytime.
